HP 15s-du0094tu OpenCore EFI
macOS Ventura 13 | OC 1.0.x | SMBIOS: MacBookPro15,2

BEFORE FIRST BOOT:
1. Generate SMBIOS with GenSMBIOS (MacBookPro15,2)
2. Replace placeholder values in config.plist:
   - SystemSerialNumber
   - MLB (Board Serial)  
   - SystemUUID
   - ROM (your MAC address)

ALL .PLACEHOLDER files need to be replaced with actual binaries.
See HP-15s-Hackintosh-Guide.md for download links.
